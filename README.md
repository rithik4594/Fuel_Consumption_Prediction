# Fuel Consumption Prediction

A Machine Learning project that predicts vehicle fuel consumption based on various vehicle specifications and performance parameters.

## Project Overview

This project uses machine learning algorithms to analyze vehicle data and predict fuel consumption. The model helps understand how factors such as engine size, cylinders, and emissions influence fuel efficiency.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection and engineering
- Machine Learning model training
- Fuel consumption prediction
- Model performance evaluation
- Data visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset

The dataset contains vehicle-related attributes such as:

- Engine Size
- Number of Cylinders
- Fuel Type
- Transmission Type
- CO₂ Emissions
- Fuel Consumption (Target Variable)
